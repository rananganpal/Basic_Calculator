# 🧮 3D Math Symbols Calculator

A **visually stunning calculator** with **3D rotating math symbols** in the background!  
Built with **HTML, CSS, and JavaScript**, it combines a functional calculator with an interactive 3D animated effect. Perfect for math enthusiasts and creative coding demos. ✨

---

## 🌟 Features

- 🎨 **3D Animated Background:** Rotating math symbols like `+`, `-`, `∑`, `π`, and more.  
- 🖱️ **Interactive Calculator:** Supports basic arithmetic operations (`+`, `-`, `*`, `/`, `%`).  
- 🖤 **Dark Theme:** Sleek modern UI with glowing math symbols.  
- ⚡ **Responsive Design:** Works on any screen size.  
- 💡 **Fun and Educational:** Combines math learning with visual effects.
